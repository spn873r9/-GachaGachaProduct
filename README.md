# Gacha Feed Pages

このPublicリポジトリは、iOSアプリが取得する静的JSONだけを配信するためのGitHub Pagesリポジトリです。

開発コード、内部ドキュメント、ユーザー情報、秘密情報は置きません。

## 配信ファイル

```text
.nojekyll
v1/manifest.json
v1/latest.json
v1/manufacturers.json
v1/releases/*.json
v1/products/*.json
```
